          Open PCPACK / PS2PACK — parse a .PCPACK (PC) or .PS2PACK (PlayStation 2) file. Format is auto-detected from extension. PS2PACK uses .PS2* / .TM2 naming, and the field at directory+0x70 (file offset 0xB4 with the default directory_offset=0x30) is parsed as a 12-byte-per-entry table of extra texture data and preserved verbatim on rebuild.

          Load Dictionary — text file with lines of 0xHHHHHHHH name for hash → name resolution.

          Add Replacement Files — pick files whose names match displayed filenames (e.g. 0x12345678.DDS or hero_body_diff.DDS); they queue as replacements at the same index.

          Build — repack with replacements at their original indices (mirrors do_import).

          Reimport — pick a folder; existing entries are updated by hash+type, new entries are added, list is sorted by (type, hash), TL entries for textures/meshes/skeletons/etc are added or remapped, PCANIM clips are re-parsed and anim_locations is rebuilt (mirrors do_reimport_from_folder).

          You can also drag & drop .PCPACK, .TXT dictionary, or replacement files onto the window.
